nanjinglug.org
===============

## 环境

基于 Jekyll & GitHub Pages 搭建

Theme: [mark-reid][mark-reid]

## 协作

#### 如果想更新网站代码(例如更新样式主题，添加板块，更新布局等等)：

* Fork 这个 Git 库
* 在你的 Git 库中做修改并 commit 你的修改
* 向这里发送 Pull Request

#### 如果想添加文章(例如聚会公告，聚会记录):

* Fork [articles] Git 库
* 在你的 Git 库中做修改并 commit 你的修改
* 向 [articles] 发送 Pull Request

## 贡献

* [代码维护][site-code]
* [文章贡献][articles-contrib]
